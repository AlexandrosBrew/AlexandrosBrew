## About Me

I am a Robotics and Artificial Intelligence student with a strong interest in building intelligent, interactive, and accessible robotic systems. My work focuses on combining **computer vision**, **machine learning**, **embedded systems**, and **robotic control** to create practical prototypes that connect perception with real-world actuation.

My main technical interests include:

- Gesture-based human-robot interaction
- Real-time computer vision systems
- Robotic arm control and inverse kinematics
- Machine learning for perception and classification
- Arduino and embedded actuation systems
- 3D-printed robotic prototypes
- Python-based robotics and AI applications

I enjoy developing projects that move beyond isolated algorithms and demonstrate complete working systems, from sensing and processing through to control, communication, and physical output.

## Featured Projects

### Gesture-Based Robot Control Honours Project

A computer-vision-based human-robot interaction system that uses real-time hand tracking and gesture recognition to control a 3D-printed 3DoF robotic arm.

**Key technologies:** Python, OpenCV, MediaPipe, Arduino, PID control, inverse kinematics, serial communication, 3D printing

**Project highlights:**

- Real-time hand tracking using MediaPipe Hands
- Continuous and discrete gesture-based robot control
- Left-hand and right-hand control mappings
- PID-based smoothing for more stable joint movement
- Custom Python-to-Arduino serial communication protocol
- 3D-printed robotic arm actuated using servo motors
- Evaluation focused on usability, accessibility, reliability, and learnability

**Repository:** [Gesture-Based-Robot-Control-Honours-Project](https://github.com/AlexandrosBrew/Gesture-Based-Robot-Control-Honours-Project)

---

### AirCanvas

A real-time computer vision drawing application that allows users to draw on a virtual canvas using hand gestures captured through a webcam.

**Key technologies:** Python, OpenCV, MediaPipe, TensorFlow/Keras, NumPy

**Project highlights:**

- Touch-free virtual drawing using index finger tracking
- Live webcam overlay with digital canvas rendering
- Gesture-aware drawing control
- Colour selection toolbar
- Canvas clearing and screenshot saving
- Modular hand-tracking and gesture-recognition components

**Repository:** [AirCanvas](https://github.com/AlexandrosBrew/AirCanvas)
